# Election Guide AI 🗳️

## Problem
Many citizens, especially first-time voters, find it difficult to understand the election process, timelines, and steps.

## Solution
Election Guide AI is an interactive assistant that simplifies the election process using a step-by-step approach and smart responses.

## Features
- Step-by-step election process guide
- Interactive question-answer assistant
- Quick help buttons for common scenarios
- Simple and user-friendly UI

## How It Works
Users can:
1. View election steps clearly
2. Click quick help options
3. Ask questions about elections

## Tech Stack
- HTML
- CSS
- JavaScript

## Assumptions
- This is a demo project
- No real-time government API is used
