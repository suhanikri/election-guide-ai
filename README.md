# 🗳️ Election Guide AI

## 🚀 Overview
Election Guide AI is an interactive, AI-powered assistant designed to simplify the election process for citizens, especially first-time voters. It provides step-by-step guidance, answers queries in simple language, and creates an intuitive experience for understanding how elections work.

---

## ❗ Problem
Many people find the election process confusing:
- How to register?
- What documents are required?
- Where is my polling booth?
- What happens on voting day?

This lack of clarity leads to hesitation and reduced participation.

---

## 💡 Solution
Election Guide AI solves this by:
- Providing a **clear step-by-step election journey**
- Offering an **AI-powered chat assistant**
- Giving **instant answers to common voter questions**
- Making the experience simple, interactive, and user-friendly

---

## ✨ Key Features
- 📌 Step-by-step election process visualization  
- 💬 AI-powered assistant using Google Gemini API  
- ⚡ Interactive chat interface with typing animation  
- 🎯 User-friendly and beginner-focused design  
- 📱 Responsive and clean UI  

---

## 🧠 How It Works
1. Users view the election steps clearly on the homepage  
2. Users ask questions in the chat interface  
3. Gemini AI processes queries and returns simple explanations  
4. Users get instant, easy-to-understand guidance
